JANUA

Accelerate your 3D scene using Janua!

Janua is an automatic portal and cell generator used to perform occlusion culling in 3D scenes. Janua calculates the Potentially Visible Set (PVS) of the scene and saves it into a file that is queried at runtime.

Janua maintained by gigc
